## v1 → v2 변경 사항

v1 기능 구현 완료 후 UX 검토, 신규 기능 기획, 보안 감사 결과를 반영하여 다음 3개 영역이 추가되었습니다.

- **UX 수정**: 메타데이터 오기재, 접근성, 클릭 어포던스 등 7건
- **P0 신규 기능**: 검색, 정렬, 독서 통계 시각화
- **보안 수정**: High 2건, Medium 3건, Low 2건
