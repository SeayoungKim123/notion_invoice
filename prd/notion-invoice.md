# PRD: Notion 기반 견적서 생성 및 PDF 다운로드

- **작성일:** 2026-03-28
- **상태:** Confirmed

---

## 1. 개요 (Overview)

**한 줄 요약**
Notion에 입력한 견적 데이터를 웹 페이지에서 견적서로 자동 렌더링하고, PDF로 다운로드할 수 있는 기능

**배경 및 문제 정의**
견적서를 매번 수동으로 작성하거나 별도 툴(Excel, Word 등)로 만드는 방식은 비효율적이고 버전 관리가 어렵다. Notion을 데이터 소스로 활용하면 익숙한 환경에서 데이터를 입력하고, 웹에서 일관된 형식의 견적서를 자동 생성할 수 있다.

**목표 및 기대 효과**
- 견적서 작성 시간 단축
- 일관된 브랜드/포맷의 견적서 제공
- PDF 다운로드로 클라이언트 전달 편의성 향상

---

## 2. 사용자 (Users)

**타겟 사용자**
Notion을 업무 툴로 사용하는 프리랜서, 소규모 사업자, 에이전시 담당자

**사용자 페르소나**

| 항목 | 내용 |
|---|---|
| 이름 | 김대표 |
| 역할 | 소규모 에이전시 대표 |
| 목표 | 클라이언트에게 빠르고 깔끔한 견적서를 전달하고 싶다 |
| 불편함 | 매번 Excel 파일을 열고, 수정하고, PDF로 저장하는 과정이 번거롭다 |

**사용자 여정**

1. Notion 데이터베이스에 견적 항목(품목, 수량, 단가, 메모 등) 입력
2. 웹 앱에서 해당 견적서 URL 접속
3. 렌더링된 견적서 확인
4. PDF 다운로드 버튼 클릭 → PDF 저장

## 3. 기능 요구사항 (Functional Requirements)

### 우선순위

| 우선순위 | 기능 |
|---|---|
| Must | Notion API 연동으로 견적 데이터 불러오기 |
| Must | 웹 페이지에 견적서 렌더링 |
| Must | PDF 다운로드 |
| Should | 견적서 URL 공유 (슬러그 기반) |
| Should | 견적서 상태 표시 (초안 / 확정 / 만료) |
| Could | 견적서 미리보기 (인쇄 레이아웃) |
| Could | 다국어 지원 (한/영) |

### 기능 상세

**Notion API 연동**
- Notion Integration Token으로 특정 데이터베이스 접근
- **단일 DB 구조:** 견적서 1개 = Notion 페이지 1개. 품목 목록은 텍스트 또는 멀티셀렉트 필드로 관리
- 불러올 필드: 견적명, 고객사, 발행일, 유효기간, 품목 목록(품목명·수량·단가·비고), 합계, 세금, 총액, 담당자

**견적서 렌더링**
- URL 패턴: `/[slug]` (Notion 페이지 ID 기반 랜덤 슬러그, 인증 없이 공개 접근 가능)
- 구성 요소: 헤더(로고·회사정보 하드코딩), 고객사 정보, 품목 테이블, 금액 합계, 푸터(담당자·유효기간)
- 회사 정보(로고, 상호, 연락처 등)는 코드에 하드코딩으로 관리

**PDF 다운로드**
- 클라이언트 사이드 생성: html2pdf.js 사용
- 파일명: `견적서_[고객사]_[발행일].pdf`
- 인쇄 최적화 레이아웃 적용

**엣지 케이스**
- Notion 데이터 없거나 비공개 페이지일 경우 → 404 안내 페이지
- 품목이 0개일 경우 → 빈 견적서 경고 표시
- 네트워크 오류 시 → 재시도 안내

## 4. 비기능 요구사항 (Non-Functional Requirements)

- **성능:** 견적서 페이지 로딩 3초 이내 (Notion API 응답 포함)
- **보안:** Notion API 키는 서버 환경변수로만 관리, 클라이언트에 노출 금지
- **접근성:** 시맨틱 HTML 사용, 인쇄 미디어 쿼리 적용
- **지원 환경:** Chrome, Safari, Edge 최신 버전 / 모바일 뷰 대응

## 6. 범위 정의 (Scope)

**In Scope**
- Notion 데이터베이스 연동
- 견적서 웹 렌더링
- PDF 다운로드

**Out of Scope**
- 견적서 내 직접 편집 기능 (Notion에서 편집)
- 이메일 발송 기능
- 전자서명
- 결제 연동

**향후 고려 사항**
- 견적서 템플릿 커스터마이징
- 고객사 별 전용 링크 + 열람 알림
- 전자서명 및 승인 플로우
